# Python Special Methods (Magic/Dunder Methods)

This repository contains a Jupyter Notebook demonstrating some core Python special methods. 

### Covered Methods:
1. __init__ - Object Constructor
2. __del__ - Object Destructor
3. __str__ & __repr__ - String Representations
4. __len__ - Length Enforcement
5. __eq__ - Equality Evaluation
6. __add__ & __sub__ - Operator Overloading
7. __getitem__ & __setitem__ - Index Access & Assignment
8. __contains__ - Membership Testing
9. __call__ - Callable Instances
10. __iter__ & __next__ - Iteration Support
11. __bool__ - Boolean Truthiness
