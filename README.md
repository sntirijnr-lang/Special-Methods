Python Special Methods (Magic/Dunder Methods)
This repository contains a Jupyter Notebook demonstrating some core Python special methods.

Methods to be displayed:
del - Object Destructor
call - Callable Instances
len - Length Enforcement
eq - Equality Evaluation
add & sub - Operator Overloading
getitem & setitem - Index Access & Assignment
contains - Membership Testing
str & repr - String Representations
iter & next - Iteration Support
bool - Boolean Truthiness
init - Object Constructor
